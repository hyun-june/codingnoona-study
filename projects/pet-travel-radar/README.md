# 펫 트래블 레이더

한국관광공사 오픈 API를 활용한 반려동물 동반 여행 추천 서비스 제안서입니다.

## 1. 서비스 기획 배경 및 필요성

반려동물을 가족처럼 여기는 인구가 늘어나면서 여행 방식도 함께 변화하고 있습니다. 하지만 반려동물과 함께 이동할 수 있는 관광지, 산책로, 카페, 식당, 숙소 정보를 찾는 과정은 여전히 번거롭습니다. 사용자는 여러 블로그, 지도 리뷰, 개별 업체 안내를 직접 확인해야 하며, 실제 방문 시에는 동반 가능 여부, 야외석 유무, 목줄 규정, 크기 제한 등 세부 조건이 달라 혼란을 겪기 쉽습니다.

기존 여행 정보 서비스는 관광지를 목록이나 검색 결과로 보여주는 데 집중되어 있어, 사용자의 현재 위치와 이동 맥락을 반영한 즉각적인 추천 경험이 부족합니다. 특히 여행 중에는 사용자가 긴 설명을 읽기보다 "지금 내 주변에서 반려동물과 갈 수 있는 곳"을 빠르게 알고 싶어 합니다.

따라서 공공 관광 데이터를 위치 기반 모바일 경험으로 재구성하여, 반려동물 보호자가 여행 중 더 쉽고 안전하게 장소를 선택할 수 있는 서비스가 필요합니다. 본 서비스는 한국관광공사의 반려동물 동반 여행 데이터를 활용해 사용자의 현재 위치 주변 장소를 실시간으로 감지하고, 반경 안에 들어온 장소를 추천하는 방식으로 정보 탐색의 부담을 줄이고자 합니다.

## 2. 서비스 개요

펫 트래블 레이더는 포켓몬 고처럼 사용자의 이동과 주변 반경을 화면에 시각화하고, 반려동물과 함께 방문할 수 있는 관광 정보를 추천하는 React Native 기반 모바일 서비스입니다.

사용자가 앱을 실행하면 현재 위치를 중심으로 탐색 반경이 표시됩니다. 주변에 반려동물 동반 가능 장소가 있으면 지도 또는 레이더 화면에 마커가 나타나고, 사용자가 해당 장소에 가까워질수록 추천 카드와 상세 정보가 제공됩니다. 데모 앱에서는 실제 GPS와 API 연동 전 단계로 목데이터를 사용해 위치 이동, 반경 스캔, 장소 추천, 상세 정보 확인 흐름을 구현했습니다.

주요 기능은 다음과 같습니다.

- 현재 위치 기반 반려동물 동반 가능 장소 탐색
- 탐색 반경 안 장소 자동 감지
- 장소 카테고리별 마커 표시
- 추천 카드 제공
- 장소 상세 바텀시트 제공
- 반려동물 동반 조건, 주소, 운영 상태, 추천 태그 표시
- 향후 실제 GPS, 지도, 한국관광공사 API 연동 가능 구조

## 3. 데이터 활용 방안

본 서비스는 한국관광공사 오픈 API 중 반려동물 동반 여행과 관련된 관광 정보를 핵심 데이터로 활용합니다. API에서 제공되는 장소명, 주소, 좌표, 분류, 소개 정보, 이미지, 연락처, 운영 정보 등을 모바일 앱에서 사용하기 좋은 형태로 가공합니다.

데이터 활용 방식은 다음과 같습니다.

- `좌표 데이터`: 사용자 현재 위치와 장소 사이의 거리를 계산하고, 탐색 반경 안에 들어온 장소를 감지합니다.
- `장소명 및 분류`: 카페, 관광지, 산책로, 식당, 숙소 등 사용자가 빠르게 이해할 수 있는 카테고리로 표시합니다.
- `주소 및 위치 정보`: 장소 상세 정보와 길찾기 연동에 활용합니다.
- `반려동물 동반 정보`: 실내 동반 가능 여부, 야외석, 목줄 필수, 체중 제한, 사전 문의 필요 여부 등 방문 전 확인해야 할 조건으로 제공합니다.
- `이미지 및 소개 정보`: 장소 상세 화면에서 사용자가 방문 여부를 판단할 수 있도록 시각 자료와 설명을 제공합니다.
- `운영 정보`: 현재 이용 가능 여부, 추천 방문 시간, 휴무 가능성 등을 안내하는 데 활용합니다.

현재 데모의 목데이터는 `data/petTravelPlaces.ts`에 정의되어 있으며, 실제 API 연동 시 동일한 데이터 구조로 변환해 화면 컴포넌트를 유지한 채 데이터 소스만 교체할 수 있도록 설계했습니다.

## 4. 서비스 발전 방향

1차 데모는 목데이터 기반의 위치 탐색 경험을 검증하는 데 목적이 있습니다. 이후 실제 한국관광공사 API와 GPS 기능을 연동하면 사용자 위치를 기준으로 실시간 장소 추천이 가능해집니다.

향후 발전 방향은 다음과 같습니다.

- 실제 GPS 기반 현재 위치 추적
- 지도 SDK 연동을 통한 실제 지도 화면 제공
- 한국관광공사 API 실시간 조회 및 캐싱
- 반려동물 크기, 성향, 이동 수단에 따른 맞춤 추천
- 산책 중심, 카페 중심, 숙박 중심 등 여행 코스 자동 생성
- 장소 상세 화면에 이미지, 전화번호, 운영시간, 홈페이지, 데이터 출처 표시
- 사용자 즐겨찾기 및 여행 코스 저장
- 팀원 또는 가족과 여행 코스 공유
- 혼잡도, 날씨, 계절 축제 정보와 결합한 추천 고도화
- AR 카메라 모드 또는 게임형 리워드 요소 확장

장기적으로는 단순 관광 정보 앱이 아니라, 반려동물 보호자의 이동 맥락과 공공 관광 데이터를 연결하는 개인화 여행 가이드로 발전할 수 있습니다.

## 5. 데모 실행 방법

```bash
npm install
npm run start
```

Expo Go 앱으로 QR 코드를 스캔하면 모바일 기기에서 데모를 확인할 수 있습니다.
