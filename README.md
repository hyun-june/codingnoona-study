# Frontend Monorepo

개인 학습 코드와 이전 프로젝트를 체계적으로 관리하기 위한 프론트엔드 모노레포입니다.

여러 레포로 분산되어 있던 코드를 하나의 저장소로 통합하여 학습 과정과 코드 흐름을 함께 관리하는 것을 목표로 합니다.

---

## Structure

```txt
.
├── study/           # 강의 및 커리큘럼 기반 학습 코드
│   ├── coding-noona/
│   └── lecture/
├── portfolio/       # 포트폴리오 사이트 코드
├── archive/         # 이전 프로젝트 및 참고용 코드
└── projects/        # 프로젝트 단위 실험 및 공모전 데모
    └── pet-travel-radar/
```

---

## About

이 레포는 다음 기준으로 코드를 분류합니다.

- **study**: 강의 및 학습 기반 코드
- **portfolio**: 포트폴리오 사이트 코드
- **archive**: 과거 프로젝트 및 참고용 코드
- **projects**: 프로젝트 단위 실험, 데모 앱, 공모전 준비 코드

---

## Projects

### pet-travel-radar

한국관광공사 오픈 API 활용 공모전을 위한 Expo 기반 React Native 데모 앱입니다.

반려동물 동반 여행 데이터를 위치 기반 레이더 경험으로 재구성하여, 사용자가 이동할 때 주변 반경 안의 반려동물 동반 가능 관광지, 산책로, 카페, 숙소 등을 추천하는 콘셉트를 검증합니다.
